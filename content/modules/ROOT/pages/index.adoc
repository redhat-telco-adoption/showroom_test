= OpenShift Troubleshooting Workshop
include::_attributes.adoc[]

Welcome to the *OpenShift Troubleshooting Workshop*! In this hands-on workshop, you will diagnose and fix 19 broken application environments running on OpenShift. Each lab presents a realistic failure scenario that platform engineers and developers encounter in production.

== What you will learn

By the end of this workshop you will be able to:

* Systematically diagnose common OpenShift application failures
* Use `oc` CLI commands and the Web Console to investigate issues
* Fix configuration, storage, networking, security, and resource problems
* Build confidence troubleshooting production OpenShift clusters

== How to use this guide

Before you begin, make sure the URL of this workshop guide includes your user parameters. Your instructor will provide a URL like:

[source,text]
----
https://<workshop-url>?USER=user1&PASSWORD=openshift-workshop&CLUSTER_SUBDOMAIN=apps.cluster.example.com
----

All commands in this guide will automatically substitute your username and cluster details when the URL parameters are set correctly.

Start with xref:00-setup.adoc[Environment Setup] to verify your access, then work through the labs at your own pace.

TIP: Each lab includes progressive hints and a full solution. Try to solve each one before looking at the hints!
