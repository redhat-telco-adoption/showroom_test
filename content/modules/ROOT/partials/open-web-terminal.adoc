To open the web terminal in the OpenShift web console:

. In the web console header, click the *>_* (terminal) icon to the right of the search bar.
+
// TODO: Add screenshot of web console header with terminal icon highlighted

. A terminal panel appears at the bottom of the browser window.

. The terminal is pre-authenticated to the cluster with your user credentials.

NOTE: If the terminal takes more than 30 seconds to appear, refresh the page and try again.
